TO UPDATE REMOTE REPO W.R.T LOCAL (to copy files from local repo to remote):
the three commands are:
add
commit
push
templates for add:
$ git add<path>
examples:
$ git add.
store all files from the current folder and all files from sub folder also
 
$ git add ./folder
stages all fles from the sub folder named "programs" and all files from its sub folder also
 
templates for commit:
$ git commit -m "<revelent message>"
 
example:
$ git commit -m "program for implementing binary search using recursin is added. though it is bad 
way to slove the problem using recurssion"
  
  
template for push:
$ git push
if you are sure of pushing ito the main branch alternatively if we have no other branch
  
$git push origin backend
if we are sure enough to push to a speific branch provided we are in the specific branch
  
TO UPDATE REMOTE REPO W.R.T LOCAL (to copy files from local repo to remote):
  
$ git pull origin main
note that if any files are deleted in the remote repo, these files will also deleted in the local repo
